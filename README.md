# YoungLimit

Young Limit is a simple yet powerful tool oriented android application that helps you see the real time speed limitation while driving. 🚗💨

## Features

### Implemented
- Real time speed limitation ✅
- Dark theme 🌙
- Material UI support 🎨

### Planned
- Speedometer 🏎️
- Speed limit warning ⚠️
- Speed limit alarm ⏰
